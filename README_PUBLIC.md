# 学生多维画像分析系统（对外发布版）

面向高校学生管理场景的智能分析与干预报告系统。

## 项目定位

系统融合多源校园行为数据、风险预测模型与大语言模型推理能力，自动生成可执行的个性化学生发展建议，帮助辅导员从“看数据”走向“做决策”。

## 核心能力

- 多源数据融合：统一整合教务、考勤、学习、上网、体测、就业等异构数据
- 画像建模：10个核心画像维度 + 1个群体聚类画像
- 风险预测与解释：基于 XGBoost + SHAP 的可解释风险归因
- 知识增强生成：KERAG 四层提示架构驱动深度个性化报告
- 多提供商 LLM：支持 DeepSeek、智谱、Moonshot、OpenAI、Claude、Ollama

## 架构总览

1. 数据层：多源校园行为数据采集与标准化
2. 建模层：画像构建、风险预测、异常检测、群体聚类
3. 解释层：逐实例 SHAP 贡献值归因
4. 生成层：知识检索注入 + 报告模板约束 + LLM 输出

## 主要产出

- 学生个体画像与综合风险等级
- 关键风险因子与保护因子解释
- 分阶段干预建议（阻断期 / 重构期 / 发展期）
- Markdown 报告与配套可视化图表

## 典型报告类型

- 学习动机归因分析
- 网络依赖风险诊断
- 学困生逆袭路径建议
- 职业发展路径规划
- 综合预警干预报告

## 数据与规模（当前样例）

- 覆盖约 2500 名学生
- 10 份画像结果 CSV + 1 份群体聚类结果
- 聚类输出 5 类群体特征

## 价值亮点

- 提升预警时效：由事后识别转向事前干预
- 提高辅导效率：从人工逐条研判变为自动化诊断
- 增强沟通质量：报告兼顾数据依据、心理机制和行动路径
